# Platform Safe Layout

Use this skill to ensure the concept survives platform cropping and overlays.

## Rules
- Story layouts need strong vertical composition and protected top/bottom zones.
- Feed graphics should read immediately as thumbnails and still reward full-size viewing.
- Leave breathing room for on-image text when exact text is required.
- Keep focal points away from crop-risk edges unless the crop itself creates the effect.

## Aspect Ratio Defaults
- story: 9:16
- portrait: 4:5
- square: 1:1
- landscape: 16:9
- cover: 3:2
