# SKILL.md — Premium Project Launch Prompting

## Purpose

Use this skill to generate prompts for **premium real-estate project launch social posts**.

The goal is to create prompts that produce images which feel:
- premium
- launch-worthy
- project-led
- brand-worthy
- social-media-ready
- designed like a polished poster, not a raw render or brochure flyer

This skill is for:
- project launches
- new tower launches
- new phase launches
- premium announcement posts
- launch campaign hero posts

---

## Core objective

A good project launch prompt should create:

**real project truth + strong launch hierarchy + premium poster treatment**

The image should:
- preserve the actual building identity
- make the project the hero
- feel like a designed launch poster
- remain clean, modern, and memorable

---

## Non-negotiable rules

1. **BRIEF OVERRIDES THIS SKILL: If the brief explicitly specifies mood, lighting, style, colors, or atmosphere, that ALWAYS takes priority over any preference in this skill. For example: if brief says "Night view", use night lighting even if this skill suggests daylight.**
2. **Use the real project image as truth anchor**
3. **Preserve exact building identity**
4. **Keep the project as the hero**
5. **Design it as a social poster, not a plain render**
6. **Use a clear launch-message zone**
7. **Keep hierarchy strong and clean**
7. **Avoid brochure clutter**
8. **Do not overfill with legal/compliance unless required in brief**

---

## What good outputs should feel like

- premium project launch poster
- polished Instagram / social media launch creative
- architectural, aspirational, and brand-worthy
- modern and restrained
- premium builder campaign quality

They should **not** feel like:
- raw building render
- generic brochure page
- noisy sales flyer
- cluttered compliance ad
- floating fantasy environment unless intentionally requested
- overdecorated luxury cliché

---

## Asset rules

- Use the supplied project image as the identity reference: preserve tower silhouette, massing, facade rhythm, podium shape, balcony language, and recognizable proportions.
- If the supplied image shows a completed building exterior, preserve the final identity.
- If the briefed launch calls for a mood different from the supplied image, add mood direction but preserve identity.
- Do not distort or reimagine the project unless explicitly requested.

---

## Prompting workflow

### 1. Lock the truth
Always mention:
- supplied project image / reference
- preserve exact identity
- preserve silhouette, massing, facade rhythm, podium proportions, balcony language, overall character

### 2. Choose framing
Pick one:
- full-building hero
- right-heavy tower crop
- centered tower hero
- inset / framed project visual
- cutout building on clean background
- dusk premium hero
- lifestyle-leaning but still project-led

### 3. Define design treatment
The image should read as:
- premium launch poster
- designed social-media post
- polished Canva-style launch creative
- editorial real-estate announcement

Use:
- strong message zone
- clear hierarchy
- premium containment
- one or two design gestures only

Do **not** ask for:
- dashboard UI
- brochure density
- too many badges
- too many info boxes
- random decorative clutter

### 4. Define mood
**IMPORTANT: These are defaults only - if the brief specifies different mood/lighting, ALWAYS follow the brief.**

Prefer (when brief doesn't specify):
- clean daylight
- soft premium daylight
- quiet dusk / early evening
- premium neutral mood
- calm aspirational atmosphere

Avoid:
- overdramatic fantasy lighting
- fake lens flare
- loud sunset overload
- aggressive luxury shine

## Brand Colors & Typography (MANDATORY)
- Apply the brand palette (primary, secondary, accent, neutrals) to typography, overlay elements, divider lines, and small accents.
- Use the brand's exact headline font family for branded text and headlines.
- Typography must feel premium and integrated with the launch imagery - NOT like a sticker on top.
- Brand colors should unify the launch scene with the text overlays.
- When describing text in the scene, use the brand's exact font style (e.g., "bold Gotham sans-serif branded text").

---

## Text / Messaging Rules (MANDATORY)

Launch posts usually include a small amount of launch messaging.

Use:
- exact launch copy from the brief, if provided
- otherwise use safe generic launch messaging

Typical text layers may include:
- project name
- launch headline
- short support line
- unit-type or location label only if provided
- brand name / project brand lockup

Do not invent:
- prices
- possession claims
- legal claims
- unit types not provided
- location claims not provided
- RERA details not provided
- phone numbers not provided

If no brief text is given, keep messaging generic and premium, for example:
- Now Launched
- Introducing [Project Name]
- A new address begins here
- Crafted for elevated living
- A rare address, now launched
- Premium living, thoughtfully introduced

---

## Logo Integration (when enabled)
- Logo must integrate seamlessly with the project imagery - NOT look like a sticker on top.
- Never place logo on a white/solid background - use transparent logo that blends naturally with the scene.
- Position subtly in a footer corner or signature zone where it contrasts naturally with the image.
- The logo should feel like part of the design, not an afterthought.

## Default prompt template

```text
Create a premium real-estate project launch social poster using the supplied project image as the truth anchor. Preserve the exact building identity, silhouette, massing, facade rhythm, podium proportions, balcony language, and overall architectural character. The project should remain the hero.

The result should read as a designed launch poster rather than a raw render or brochure page. Add a refined graphic design treatment with clear hierarchy, a strong launch-message zone, clean visual containment, and a polished premium finish. The design should feel modern, architectural, and social-media-ready, with restraint and clarity.

Keep the overall composition premium, aspirational, and calm. Use daylight, soft premium light, or quiet dusk depending on the intended mood. The post may include launch messaging taken from the brief, or safe generic launch text when no exact copy is provided.

Avoid brochure clutter, fake offers, compliance-heavy overload unless required, loud gradients, fantasy lighting, generic stock styling, or any distortion of the project's true identity.
```

## Variation levers

To avoid repetitive launch posts, vary these:

**Framing:**
- full-building hero
- right-heavy tower crop
- centered tower
- inset project visual
- cutout building on clean field

**Design intensity:**
- ultra-minimal luxury
- bold launch panel
- editorial white-space layout
- stronger social-template feel

**Mood:**
- clean daylight
- soft neutral premium light
- calm dusk launch mood

**Composition gesture:**
- sculpted panel
- one bold block
- framed visual
- quiet open sky
- isolated building with large whitespace

---

## Quick repair rules

**If the output feels too plain:**
- Add: designed launch poster, stronger message zone, premium hierarchy, one clear graphic gesture, polished social-media composition

**If the output feels too cluttered:**
- Add: restrained design treatment, clean containment, premium whitespace, avoid brochure density

**If the building identity is drifting:**
- Reinforce: preserve exact project identity, preserve silhouette, massing, facade rhythm, podium proportions, use supplied image as truth anchor

**If it feels too salesy:**
- Add: premium editorial launch feel, calm aspirational mood, avoid flyer-like treatment, avoid excessive badges and text blocks

**If it feels too generic:**
- Change: framing, mood, one compositional gesture, panel style, launch hierarchy intensity

---

- Generic brochure pages
- Dashboard UI or infographics
- Excessive badges, compliance overload
- Fake offers or prices
- Fantasy floating environments unless requested
- Loud gradients or fake lens flare
- Distorted project identity
