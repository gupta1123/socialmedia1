# Launch Post Playbook

Use this playbook for `project-launch`.

## Core job
Announce the project clearly and premium-first.

## Asset rules
- Prefer the strongest recognizable project exterior.
- Do not let unrelated amenities or community scenes take over unless the brief explicitly asks for them.

## Composition families
- Hero facade reveal
- Editorial tower crop
- Premium launch announcement with quiet overlay
- Landmark framing with disciplined footer

## Required prompt behavior
- Write it as a finished premium launch poster, not a plain architecture description.
- Use a clear poster structure: small brand/header strip, clean headline-safe zone, one restrained support/location line, dominant tower hero, and a quiet footer/signature zone.
- Let the building dominate the frame and keep the text area intentionally calm and readable.
- Use at most one subtle overlay/card system and one compact badge or supporting cue.
- Avoid brochure grids, many-amenity collage behavior, and noisy skyline clutter behind the text area.

## Brand Colors & Typography (MANDATORY)
- Apply the brand palette (primary, secondary, accent, neutrals) to ALL typography, overlay elements, and accent decorations.
- Use the brand's exact headline font family (e.g., Gotham, Helvetica Neue, etc.) for project name and headlines.
- The brand's typography mood should define the text style - premium, bold, clean hierarchy.
- Brand colors should guide the overall image mood and color grading so text feels integrated with the building/project imagery.
- When describing text in the scene, use the brand's exact font style (e.g., "bold Gotham sans-serif project name at top").

## Logo Integration (when enabled)
- Logo must integrate seamlessly with the project imagery - NOT look like a sticker on top.
- Never place logo on a white/solid background - use transparent logo that blends naturally with the architecture.
- Position subtly in a footer corner or signature zone where it contrasts naturally with the image.
- The logo should enhance brand recognition without competing with the project hero.

## Must not leak
- Construction-progress logic
- Many-amenity collage behavior
- Lifestyle crowd scenes unless explicitly requested
- Text that doesn't match brand typography style
- Logo that looks pasted on top of the scene
