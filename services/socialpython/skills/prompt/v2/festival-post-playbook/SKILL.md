# Festival Post Playbook

Use this playbook for `festive-greeting`.

## Core job
Deliver a respectful premium greeting poster.

## Asset rules
- Default to project-free festive composition unless project linkage is explicitly requested.
- If a logo is included, keep it small and secondary.

## Composition families
- Symbol-led premium greeting
- Typographic blessing with restrained motif
- Editorial ceremonial still-life
- Minimal festive poster with calm negative space

## Required prompt behavior
- Write it as a finished premium greeting poster with a clear blessing-safe zone and generous negative space.
- Keep the composition project-free by default unless the brief explicitly requests project linkage.
- Use one calm symbolic focal arrangement or one restrained ceremonial motif, not a cluttered festive montage.
- If a logo is included, keep it tiny and secondary in a footer/signature zone.
- Avoid sales overlays, property facts, floor plans, and brochure clutter.

## Brand Colors & Typography (MANDATORY)
- Apply the brand palette (primary, secondary, accent, neutrals) to the festive greeting's graphic elements, typography, and accent decorations.
- Use the brand's exact headline font family for the greeting text and any blessings - match the brand's typography mood (e.g., bold contemporary sans-serif, refined serif).
- Brand colors should unify the festive imagery with the text - the color grading should feel consistent with the brand's visual identity.
- When describing text in the scene, use the brand's exact font style (e.g., "bold Gotham sans-serif 'Happy Diwali'" or "refined serif blessing text").
- Festive colors (if different from brand palette) should complement, not override, the brand palette.

## Logo Integration (when enabled)
- Logo must integrate seamlessly with the festive scene - NOT look like a sticker on top.
- Never place logo on a white/solid background - use transparent logo that blends naturally with the festive imagery.
- Position subtly in a footer corner or signature zone - the logo should feel like part of the scene, not an afterthought.
- The logo should complement the festive mood without distracting from the blessing/greeting.

## Must not leak
- Project facts by default
- Property-ad clutter
- Loud kitsch or festival overload
- Text that doesn't match brand typography style
- Logo that looks pasted on top of the scene
