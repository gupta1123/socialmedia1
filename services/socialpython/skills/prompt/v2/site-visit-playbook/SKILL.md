# SKILL.md — Premium Site Visit Prompting

## Purpose

Use this skill to generate prompts for **premium real-estate site visit social posts**.

The goal is to create prompts that produce images which feel:
- premium
- warm and inviting
- project-led or visit-led
- social-media-ready
- designed like a polished post, not a raw on-site photo or cluttered sales flyer

This skill is for:
- site visit invitation posts
- weekend site visit creatives
- experience-the-project posts
- visit-us announcements
- walkthrough / discover-the-project posts

---

## Core objective

A good site-visit prompt should create:

**real project truth + inviting visit mood + premium poster treatment**

The image should:
- preserve the actual project identity if a project image is used
- communicate welcome, discovery, and visitability
- feel like a designed social post
- remain clean, aspirational, and credible

---

## Non-negotiable rules

1. **Use the real project image as truth anchor when provided**
2. **Preserve exact building identity**
3. **Make the visit experience feel inviting**
4. **Design it as a social poster, not a plain render**
5. **Use a clear invitation / CTA zone**
6. **Keep hierarchy strong and clean**
7. **Avoid brochure clutter**
8. **Do not invent offers, dates, or event details not in the brief**

---

## What good outputs should feel like

- premium site visit invitation poster
- polished Instagram / social media creative
- welcoming, aspirational, and brand-worthy
- architectural and experiential
- modern and restrained

They should **not** feel like:
- raw building render
- generic brochure page
- noisy event flyer
- fake crowded open-house scene
- hyper-salesy ad
- cluttered compliance layout

---

## Asset rules

- Use the supplied project image as the identity reference: preserve tower silhouette, massing, facade rhythm, podium proportions, balcony language, entrance experience, and recognizable proportions.
- If the supplied image shows the entrance or arrival experience, preserve those elements.
- If no project image is provided, create a welcoming project-led visual with appropriate mood.
- Do not distort or reimagine the project unless explicitly requested.

---

## Prompting workflow

### 1. Lock the truth
Always mention:
- supplied project image / reference if available
- preserve exact identity
- preserve silhouette, massing, facade rhythm, podium proportions, balcony language, overall character

### 2. Choose framing
Pick one:
- full-building hero
- entrance / arrival experience
- approach road + project reveal
- sales gallery / site arrival mood
- medium-wide building with warm welcoming foreground
- project-led with subtle human visit cues

### 3. Define design treatment
The image should read as:
- premium site visit poster
- designed social-media post
- polished invitation creative
- editorial real-estate visit announcement

Use:
- clear CTA / invitation zone
- clean hierarchy
- premium containment
- one or two design gestures only

Do **not** ask for:
- dashboard UI
- brochure density
- too many icons
- too many info boxes
- fake "event" clutter

### 4. Define mood
Prefer:
- clean daylight
- soft weekend daylight
- golden but restrained late-afternoon warmth
- welcoming arrival mood
- premium aspirational atmosphere

Avoid:
- dramatic fantasy lighting
- loud sunset overload
- fake crowd scenes
- hyper-staged lifestyle chaos

## Brand Colors & Typography (MANDATORY)
- Apply the brand palette (primary, secondary, accent, neutrals) to typography, overlay elements, divider lines, and small accents.
- Use the brand's exact headline font family for branded text and headlines.
- Typography must feel premium and integrated with the site-visit imagery - NOT like a sticker on top.
- Brand colors should unify the invitation scene with the text overlays.
- When describing text in the scene, use the brand's exact font style (e.g., "bold Gotham sans-serif branded text").

---

## Text / Messaging Rules (MANDATORY)

Site visit posts usually include a small amount of invitation messaging.

Use:
- exact CTA / copy from the brief, if provided
- otherwise use safe generic site-visit messaging

Typical text layers may include:
- project name
- site visit headline
- short support line
- location label if provided
- weekend / visit-us CTA if provided
- brand name / project brand lockup

Do not invent:
- dates not provided
- timings not provided
- offers not provided
- food / event promises not provided
- transport / pickup claims not provided
- phone numbers not provided
- RERA details not provided

If no brief text is given, keep messaging generic and premium, for example:
- Visit the site
- Experience the address in person
- Walk through your future address
- Discover the project up close
- Plan your site visit
- Visit us this weekend

---

## Logo Integration (when enabled)
- Logo must integrate seamlessly with the project imagery - NOT look like a sticker on top.
- Never place logo on a white/solid background - use transparent logo that blends naturally with the scene.
- Position subtly in a footer corner or signature zone where it contrasts naturally with the image.
- The logo should feel like part of the design, not an afterthought.

## Default prompt template

```text
Create a premium real-estate site visit social poster using the supplied project image as the truth anchor. Preserve the exact building identity, silhouette, massing, facade rhythm, podium proportions, balcony language, and overall architectural character. The project should remain recognisable and inviting.

The result should read as a designed site visit poster rather than a raw render or brochure page. Add a refined graphic design treatment with clear hierarchy, a strong invitation / CTA zone, clean visual containment, and a polished premium finish. The design should feel modern, welcoming, architectural, and social-media-ready, with restraint and clarity.

Keep the overall composition premium, aspirational, and warm. Use daylight, soft premium light, or restrained late-afternoon warmth depending on the intended mood. The post may include site-visit messaging taken from the brief, or safe generic invitation text when no exact copy is provided.

Avoid brochure clutter, fake offers, fake crowds, event-flyer overload, loud gradients, fantasy lighting, generic stock styling, or any distortion of the project's true identity.
```

## Variation levers

To avoid repetitive site-visit posts, vary these:

**Framing:**
- full-building hero
- entrance / arrival
- approach road reveal
- medium-wide project + landscaped foreground
- project with subtle human-scale presence

**Design intensity:**
- ultra-minimal luxury
- stronger CTA panel
- editorial white-space layout
- warm invitation-template feel

**Mood:**
- bright welcoming daylight
- soft premium afternoon
- calm golden-hour warmth
- quiet weekend visit mood

**Composition gesture:**
- one clean CTA block
- framed project visual
- sculpted panel
- quiet open sky
- entrance-led composition

---

## Quick repair rules

**If the output feels too plain:**
- Add: designed site visit poster, stronger invitation zone, premium hierarchy, one clear graphic gesture, polished social-media composition

**If the output feels too cluttered:**
- Add: restrained design treatment, clean containment, premium whitespace, avoid flyer density

**If the building identity is drifting:**
- Reinforce: preserve exact project identity, preserve silhouette, massing, facade rhythm, podium proportions, use supplied image as truth anchor

**If it feels too salesy:**
- Add: premium editorial invitation feel, calm welcoming mood, avoid flyer-like treatment, avoid excessive badges and text blocks

**If it feels too generic:**
- Change: framing, arrival mood, one compositional gesture, CTA intensity, foreground / entrance emphasis

---

## Must not leak
- Raw building renders without design treatment
- Generic brochure pages
- Dashboard UI or infographics
- Fake crowds or event staging
- Fake offers or prices
- Loud gradients or fake lens flare
- Distorted project identity
- Cluttered sales flyers
- Generic stock-visitor scenes
