# SKILL.md — Premium Amenity Spotlight Prompting

## Purpose

Use this skill to generate prompts for **premium real-estate amenity spotlight social posts**.

The goal is to create prompts that produce images which feel:
- premium
- experience-led
- amenity-focused
- social-media-ready
- designed like a polished poster, not a generic lifestyle shot

This skill is for:
- amenity spotlight posts (one amenity only)
- amenity feature highlights
- amenity experience creatives
- amenity benefit posts

---

## Core objective

A good amenity spotlight prompt should create:

**single amenity desirability + clear hierarchy + premium poster treatment**

The image should:
- make one amenity the clear hero
- communicate the lived experience
- feel like a designed social post
- remain clean, aspirational, and credible

---

## Non-negotiable rules

1. **Focus on ONE amenity only**
2. **Keep project association quiet and secondary**
3. **Design it as a social poster, not a lifestyle mood shot**
4. **Use clear poster hierarchy**
5. **Make the amenity dominant at thumbnail size**
6. **Preserve generous negative space for typography**
7. **Avoid multi-amenity collages**
8. **Do not invent amenities not available**

---

## What good outputs should feel like

- premium amenity spotlight poster
- polished Instagram / social media creative
- experiential, aspirational, and amenity-desirable
- architectural and lifestyle-connected
- modern and restrained

They should **not** feel like:
- generic lifestyle mood shot
- multi-amenity collage board
- stock photo with text overlay
- busy amenity listing
- project launch hero

---

## Asset rules

- Focus on ONE amenity only: swimming pool, clubhouse, gym, landscape deck, etc.
- Keep project association clear without turning the frame into a many-amenity board.
- If project context is needed, keep it secondary and quiet - it must never replace the amenity as the hero subject.
- Use at most one compact badge, location cue, or lifestyle-benefit note.
- If the amenity is amenities-without-images, choose a different amenity.

---

## Composition families

- Hospitality-led amenity hero
- Quiet luxury detail crop
- Amenity experience frame with clear headline zone

---

## Prompting workflow

### 1. Choose one amenity
Select ONE specific amenity:
- swimming pool
- clubhouse exterior
- clubhouse lawn / landscape deck
- outdoor gym
- sculpture plaza
- sky lounge
- yoga deck
- meditation plaza
- pantry
- co-working space

### 2. Define framing
Pick one:
- amenity-as-hero (full amenity in frame)
- detail crop (close-up luxurious detail)
- experience frame (people using amenity subtly implied)
- amenity with project context (quiet building in background)

### 3. Define design treatment
The image should read as:
- premium amenity poster
- designed social-media post
- polished amenity highlight
- editorial amenity feature

Use:
- clear headline zone
- clean hierarchy
- premium containment
- one design gesture only

### 4. Define mood
Prefer:
- clean daylight
- soft premium light
- experiential warmth
- aspirational atmosphere

Avoid:
- dramatic fantasy lighting
- generic stock photography
- busy amenity collage

## Brand Colors & Typography (MANDATORY)

- Apply the brand palette (primary, secondary, accent, neutrals) to typography, overlay accents, and any graphic elements.
- Use the brand's exact headline font family (e.g., Gotham, Helvetica Neue, etc.) for all text elements.
- Typography mood should match the brand's defined style - keep it premium, bold, and clean.
- Brand colors should guide the overall image mood so text feels integrated, not pasted on.
- When describing text in the scene, specify it using the brand's exact font style (e.g., "bold Gotham sans-serif text").

---

## Text / Messaging Rules (MANDATORY)

Amenity spotlight posts usually include amenity messaging.

Use:
- exact amenity name from the brief
- otherwise use safe generic amenity messaging

Typical text layers may include:
- amenity name as headline
- short support line describing the experience
- project name if secondary
- location or feature badge

Do not invent:
- amenities not available
- features not in the brief
- unit types
- pricing
- legal claims

If no brief text is given, keep messaging generic and premium, for example:
- Swim. Relax. Repeat.
- Your private retreat
- Where wellness meets luxury
- Built for lifestyle
- The amenity living advantage
- Elevate your daily routine

---

## Logo Integration (when enabled)

- Logo must integrate seamlessly with the image - NOT look like a sticker on top.
- Never place logo on a white/solid background - use transparent logo that blends with the scene.
- Position subtly in a footer corner or signature zone where it naturally contrasts with the image.
- The logo should feel like part of the design, not an afterthought.

## Default prompt template

```text
Create a premium real-estate amenity spotlight social poster featuring [AMENITY NAME] as the hero. Preserve the exact amenity identity, detail, and experiential quality.

The result should read as a designed amenity poster rather than a generic lifestyle mood shot. Add a refined graphic design treatment with clear hierarchy, a strong amenity-headline zone, clean visual containment, and a polished premium finish. The amenity should be dominant at thumbnail size with generous negative space for typography.

Keep the project context quiet and secondary if included. Use clean daylight, soft premium light, or experiential warmth to set the mood. The design should feel modern, architectural, and social-media-ready, with restraint and clarity.

Avoid multi-amenity collages, generic stock lifestyle, project-launch hero logic, amenity board clutter, or any distortion of the amenity's true identity.
```

## Variation levers

To avoid repetitive amenity posts, vary these:

**Framing:**
- amenity-as-hero full frame
- detail crop (close-up luxurious)
- experience frame (subtle human presence)
- amenity + quiet project context

**Design intensity:**
- ultra-minimal luxury
- stronger headline panel
- editorial white-space layout
- premium template feel

**Mood:**
- clean daylight
- soft premium afternoon
- experiential warmth
- aspirational calm

---

## Quick repair rules

**If the output feels too plain:**
- Add: designed amenity poster, stronger headline zone, premium hierarchy, one clear graphic gesture

**If the output feels too cluttered:**
- Add: restrained design treatment, clean containment, premium whitespace, avoid amenity board

**If the amenity isn't dominant:**
- Reinforce: one amenity as hero, amenity dominant at thumbnail size

**If it feels too salesy:**
- Add: premium editorial amenity feel, experiential mood, avoid direct sales language
