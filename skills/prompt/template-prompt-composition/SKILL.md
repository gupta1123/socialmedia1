# Template Prompt Composition

Use this skill whenever a reusable template is attached to the brief.

## Principles
- Templates are not fixed canvases. They are reusable direction systems.
- A template can contribute:
  - prompt scaffold
  - compositional logic
  - safe-zone expectations
  - approved use cases
  - reference image pack

## Rules
- Preserve the template's composition logic unless the brief explicitly requires a stronger deviation.
- Reuse approved use cases to keep outputs reliable.
- Blend template prompt scaffolds with current project and post-type context.
- Do not let the template overpower the current brand or project positioning.
- Treat the chosen template as the single master layout anchor; other references may support subject truth, but should not compete with its structure.

## Output Expectations
- Mention the template in prompt compilation when present.
- Use template safe-zone and layout guidance in both seed and final prompts.
- Keep the final prompt editable for reference-image workflows.
