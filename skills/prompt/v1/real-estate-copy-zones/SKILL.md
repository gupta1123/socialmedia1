# Real Estate Copy Zones

Use this skill when exact text, CTA copy, or dense headline overlays are needed.

## Principles
- Architecture and readability should not fight each other.
- Real-estate creatives often need brand mark, project name, proposition, and CTA on the same frame.

## Rules
- Reserve clear space for project name and CTA.
- Avoid placing small text over busy facades or foliage.
- For invitation or offer posts, bottom-band or side-panel structures are usually safer than floating text.
- For testimonial posts, quote readability is more important than decorative complexity.

## Output Expectations
- Mention safe zones explicitly in seed and final prompts.
- Prefer layouts that remain editable in image-plus-prompt workflows.
