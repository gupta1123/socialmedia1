# Festive Greeting Composition

Use this skill whenever the selected post type is `festive-greeting`.

## Core rule
- A festive greeting is a standalone greeting poster, not a property advertisement.

## Prompt construction rules
- Do not rely on project renders, amenities, pricing, location facts, RERA, or sales copy unless the brief explicitly asks for a project-linked festive post.
- Do not require uploaded reference images by default. Festive greetings should work from prompt-only composition and then use generated style seeds if needed.
- Write the prompt as a detailed visual production spec, not a vague concept note.
- If brand attribution is needed, use the exact brand name as plain text only in a small footer or signature line.
- Do not generate logos, monograms, emblems, icon marks, or house symbols for festive greetings.

## Required sections in the compiled image prompt
- Aspect ratio and poster orientation
- Background treatment
- Main symbolic arrangement
- Typography layout
- Decorative accents
- Style and mood
- Short negative prompt

## Visual direction
- Prefer one strong symbolic arrangement over many small festive objects.
- Preserve generous negative space so the result feels like a premium greeting card or Instagram poster.
- Keep typography elegant, sparse, and readable.
- Favor polished vector, semi-illustrative, or graphic-poster language unless the brief explicitly asks for realism.

## Cultural quality bar
- Use cues specific to the selected festival.
- Avoid mixing symbols from unrelated festivals.
- Keep the result respectful and refined rather than loud, novelty-led, or clipart-heavy.
