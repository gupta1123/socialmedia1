# Calendar Brief Normalization

Use this skill whenever a calendar item is linked to the brief.

## Objectives
- Translate a scheduled slot into a production-ready creative brief.
- Preserve the calendar item's title, objective, and timing context.

## Rules
- If the calendar objective is present, it should sharpen the prompt strategy.
- Scheduled posts should feel appropriate to the date and campaign moment.
- Calendar metadata should influence direction, not replace brand/project logic.

## Output Expectations
- Carry the calendar item title or objective into the prompt summary when useful.
- Preserve scheduling context if it changes urgency or tone.
