# Variant Critique

Use this skill to assess candidate outputs.

## Evaluate
- Brand fit
- Brief fit
- Thumbnail readability
- Text-safe composition
- Distinctiveness

## Rules
- Be explicit about what is off-brand or too generic.
- Recommend the strongest variant and explain why it is stronger.
- If all variants are weak, say so and identify the missing ingredient.
