# Missing Asset Handler

Use this skill when the asset set cannot support the ideal composition truthfully.

## Fallback options
- Prompt-led premium poster
- Logo-led editorial greeting
- Tight crop or detail treatment
- Simpler branded composition with reserved copy space

## Rules
- Prefer a truthful simpler composition over a fabricated hero image.
- Do not invent a tower, amenity, or construction state when the asset bundle cannot support it.
- For festive posts, project-free is an acceptable fallback.
- For project-led posts, downgrade ambition before sacrificing truth.
