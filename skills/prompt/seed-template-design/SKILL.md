# Seed Template Design

Use this skill for the style-seed step.

## Goal
Produce a reusable reference image that can anchor one or more finals.

## Rules
- Seeds should emphasize composition, texture, tone, and hierarchy more than finished copy.
- Keep seeds reusable: avoid overfitting them to one exact sentence unless the brief requires it.
- Design for iteration, not only immediate publishing.
- For `festive-greeting`, make the seed feel like a premium greeting-poster system with one symbolic arrangement, restrained typography, and generous negative space rather than a property-ad template.
