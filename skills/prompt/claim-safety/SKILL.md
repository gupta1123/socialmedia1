# Claim Safety

Use this skill whenever project claims, offer copy, or exact text are involved.

## Rules
- Approved claims can be reinforced.
- Banned claims must never appear.
- If exact text conflicts with brand or project restrictions, preserve the conflict as a warning signal in the prompt strategy rather than inventing unsupported messaging.
- Avoid fake scarcity, misleading pricing cues, and unverifiable location claims.

## Output Expectations
- Keep prompts claim-safe.
- Avoid overpromising in CTA phrasing.
